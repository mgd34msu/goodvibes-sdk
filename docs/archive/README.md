# Documentation Archive

This directory holds historical documentation superseded by current authoritative sources.

## `releases/0.18.x/` — Pre-0.19 per-release notes

Before 0.19.0, each release shipped a dedicated `docs/releases/<version>.md` with detailed per-release narrative. From 0.19.0 onward (per Wave S-δ), `CHANGELOG.md` at the repo root is the single canonical release log, gated at publish time by `bun run changelog:check`.

The 38 archived files here cover 0.18.14 through 0.18.51. They describe per-release changes in the form the project used at that time — they are NOT reformatted to the Keep-a-Changelog layout and should be treated as historical reference only.

For the current release log, see [`../../CHANGELOG.md`](../../CHANGELOG.md).
